A full-stack dashboard for swimming coaches and athletes, digitizing local swimming clubs' workflows with modern web technologies.

## Technical Stack

- **Frontend:** React with Tailwind CSS for responsive design
- **Visualization:** Chart.js for performance analytics
- **Backend:** Node.js with custom JWT authentication
- **Database:** PostgreSQL (migrated from MySQL for better performance)

## Key Features

- Team management and coaching tools
- Performance tracking and analytics
- Secure user authentication system
- Responsive dashboard interface
- Database optimization for query performance
